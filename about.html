About Elixra

Project Overview ✨

Elixra is a smart and user-friendly skin care tracking system designed to help users monitor their daily skin condition. It allows users to record details like acne level, dryness, oiliness, and skin type in a simple digital format.

Objectives

  • Track daily skin health records
  • Maintain organized skin history
  • Help users observe skin patterns
  • Promote healthy skincare habits
  • Provide a simple and attractive interface

Technologies Used

  • HTML for webpage structure
  • CSS for styling and animations
  • JavaScript for data storage and interactivity
  • LocalStorage for saving user entries

Future Scope 🚀

In the future, Elixra can include AI-based skin analysis, graphical progress reports, login systems, personalized skincare suggestions, and cloud database integration for better accessibility.
